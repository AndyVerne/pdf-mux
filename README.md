## Minimal Pdf viewer with split screen with React and Electron. 
<br>

![Example](example.png)
In browser example.

<br>

### Available Scripts.
In the project directory, you can run:

### `yarn run start`
### `yarn run electron-dev`
### `yarn run electron-pack`
